
## Descripción

Ejercicios pensados para apreder a manejar las exceptions en Java

### Tecnologias usadas

- Java 23

- IntelliJ

***


## Nivel 1

EJERCICIO 1:

Crea dos clases, "Producto" y "Venta", donde "Venta" tiene un método para calcular el precio total de los productos. Si no hay productos, lanza una excepción personalizada "VentaVaciaException" con un mensaje concreto.

Escribe código para generar y capturar una excepción "IndexOutOfBoundsException".

***


## Nivel 2

EXERCISE 1: 

Ejercicio en el que se comprueba que los datos introducidos por el usuario son de tipo correcto. Si no lo son, se muestra la excepción InputMismatchException o excepciones personalizadas según el caso.

***


## How to run it
Clone the repository or download the zip file, then open it in your preferred IDE.
